# shapezmono-study
非公式の Shapez Community Edition (GPLv3) を参考にした学習目的のC# / MonoGame再実装です。

>This project is not affiliated with or endorsed by tobspr-games.

### 概要
学習目的のため、試験的にコミットメッセージに感想や学んだことを記載しています。
また、参考にしてはならないコードや設計が含まれる可能性があります。
理解された上でご利用ください。
